# Documentation Archive

To archiwum przechowuje historyczne plany, PR docs, audyty, statusy i snapshoty.

Nie jest aktywnym source of truth dla agentow. Jesli dokument w archiwum jest sprzeczny z aktywnymi plikami w root `docs/`, wygrywaja:

1. `docs/CURRENT_SPRINT_PLAN.md`
2. `tasks/NEXT_TASK.md`
3. `docs/PLAN_BRAKUJACYCH_FUNKCJI.md`
4. `docs/AGENT_ORCHESTRATION.md`
5. `.github/copilot-instructions.md`

## Kategorie

| Folder | Zawartosc |
|--------|-----------|
| `agents/` | Stare checklisty agentow |
| `audits/` | Audyty i analizy |
| `features/` | Historyczne dokumenty funkcji |
| `guides/` | Stare przewodniki techniczne |
| `inventory/` | Snapshoty stanu funkcji |
| `modules/` | Historyczne plany modulow |
| `planning/` | Stare plany wykonania, beta i deployment |
| `pr/` | Historyczne PR docs |
| `status/` | Statusy zakonczonych prac |
| `strategy/` | Roadmapy i strategie historyczne |
